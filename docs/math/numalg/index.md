# 数值代数笔记

> 数值代数的核心是求解 $Ax = b$ 和 $Ax = \lambda x$。从直接法到迭代法，从三角形矩阵到特征值分解——这些方法构成了科学计算的基石。

## 知识图谱

```
Ax = b
├── 直接法 (n×n)
│   ├── 三角形方程组
│   │   ├── 前代法
│   │   └── 后代法
│   ├── 高斯消元法
│   │   ├── 无选主元 LU 分解
│   │   ├── 列主元 → PA = LU
│   │   └── 全主元 → PAQ = LU
│   ├── 平方根分解 (A > 0)
│   │   ├── Cholesky LLᵀ
│   │   └── LDLᵀ (避免开方)
│   └── 带状矩阵
│       ├── 三对角 LU → O(n)
│       └── 带状 Cholesky → O(nm²)
│
├── 误差分析
│   ├── 向量范数: ‖x‖₁, ‖x‖₂, ‖x‖∞
│   ├── 矩阵范数: 诱导范数, Frobenius, ρ(A) ≤ ‖A‖
│   ├── 条件数: κ(A) = ‖A‖·‖A⁻¹‖
│   ├── 收敛矩阵: ρ(A) < 1 ⟺ Aᵏ → 0
│   └── 舍入误差: ε_mach, 正推/反推, Kahan 求和
│
├── 最小二乘 (m×n, m > n)
│   ├── 古典方法
│   │   ├── 正规方程
│   │   └── Tikhonov 正则化
│   └── 正交化方法
│       ├── Householder 变换
│       └── Givens 旋转
│
├── 迭代法
│   ├── 收敛性理论
│   │   ├── 分裂法 M⁻¹N
│   │   └── 谱半径 ρ(G) < 1
│   ├── Jacobi 迭代
│   ├── Gauss-Seidel 迭代
│   └── SOR 松弛迭代
│
└── 共轭梯度法 (A > 0)
    ├── 最速下降法
    └── CG (n 步精确收敛 / Krylov 子空间)


特征值问题 Ax = λx
├── 任意方阵
│   ├── 幂法 (最大模特征值)
│   │   └── 反幂法 (靠近 μ 的特征值)
│   └── QR 方法 (全部特征值)
│       ├── Hessenberg 化
│       └── 带位移迭代 (Rayleigh / Francis 双位移)
│
├── 对称矩阵
│   ├── 对称 QR 方法 (三对角化 + 立方收敛)
│   ├── Jacobi 方法 (逐个消去非对角元)
│   └── 二分法 (Sturm 序列)
│
└── 大规模稀疏
    ├── Lanczos (对称)
    └── Arnoldi (非对称)
```
