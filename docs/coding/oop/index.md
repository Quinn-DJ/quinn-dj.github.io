# C++ How to Program 笔记

> "Good programmers write code that humans can understand." by *Martin Fowler*

---

此笔记为李际军老师的《面向对象程序设计》课程笔记，内容仅基于课堂PPT整理。

!!! tip "课程信息"
    - 课程名称：面向对象编程
    - 课程代码：CS2020F
    - 任课教师：李际军
    - 开课院系：计算机科学与技术学院
    - 开课学期：2025-2026 学年春夏学期
    - 学分：2.5 学分

!!! tip "成绩构成"
    课堂表现 5% + 平时作业+大程序 45% + 闭卷考试 50%。平时作业主要基于 Pintia 平台。
