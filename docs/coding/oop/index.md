# C++ How to Program 笔记

> "Good programmers write code that humans can understand." by *Martin Fowler*

---

此笔记为李际军老师的《面向对象程序设计》（Object-Oriented Programming, OOP）课程笔记，内容仅基于课堂PPT整理。共 16 讲，涵盖从 C++ 基础入门到实战项目的完整知识体系。

!!! tip "课程信息"
    - 课程名称：面向对象编程（Object-Oriented Programming, OOP）
    - 课程代码：CS2020F
    - 任课教师：李际军
    - 开课院系：计算机科学与技术学院
    - 开课学期：2025-2026 学年春夏学期
    - 学分：2.5 学分

!!! tip "成绩构成"
    课堂表现 5% + 平时作业+大程序 45% + 闭卷考试 50%。平时作业基于 Pintia 平台。
