# 数据结构与算法

这是 2025-2026 学年秋冬学期数学科学学院王何宇老师开设的《数据结构与算法》课程的笔记与解题报告。

!!! tip "课程信息"
    - 课程名称：数据结构与算法
    - 课程代码：MATH2603M
    - 任课教师：王何宇
    - 开课院系：数学科学学院
    - 开课学期：2025-2026 学年秋冬学期
    - 学分：4.0 学分
    - 参考教材：*Data Structure and Algorithm Analysis in C++*

!!! warning "考试信息"
    试卷结构：5 题（100 分）+ 附加题（10 分）。不考 C++ 语法，只要求伪代码或中文描述算法。

## 课程笔记

| 章节 | 内容 |
|:----:|------|
| [第一章：算法分析](chap-01.md) | Big-O、递推关系、主定理、数学归纳法、复杂度速查 |
| [第二章：表、栈与队列](chap-02.md) | List ADT、数组/链表实现、栈与队列操作、STL 容器 |
| [第三章：树](chap-03.md) | 二叉树、BST、AVL（旋转）、B-Tree、Splay Tree |
| [第四章：散列](chap-04.md) | 散列函数、分离链接法、开放定址法、再散列 |
| [第五章：优先队列与堆](chap-05.md) | 二叉堆（上滤/下滤/buildHeap）、堆排序前置、d-堆 |
| [第六章：排序](chap-06.md) | 插入/希尔/堆/归并/快速排序、稳定性、复杂度对比 |
| [第七章：图论算法](chap-07.md) | 拓扑排序、Dijkstra/Bellman-Ford/Floyd、Prim/Kruskal、并查集 |
